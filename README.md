# Simple Tic Tac Toe

An interactive Tic-Tac-Toe game built with React that includes game timeline and time-travel features.

## Features

* Classic 3x3 Tic-Tac-Toe gameplay
* Complete move history
* Jump to any previous game state
* Automatic win detection

## About

This project was built following the official React tutorial to learn core React concepts including state management, component composition, and immutability.

## Installation
```bash
git clone https://github.com/yourusername/react-tic-tac-toe-with-timeline.git
cd react-tic-tac-toe-with-timeline
npm install
npm start
```

Open http://localhost:3000 to play the game.

## How to Play

* Click on empty squares to place your mark
* X goes first
* Get 3 in a row to win
* Use the move history to jump back to any previous state
